[공지사항]
- 데이터베이스 설계
Notice 데이터베이스 생성
User 테이블 -> 회원정보
Content 테이블 -> 공지사항 내용
구조: Notice(i), Noticeimpl, Start

- 기능
1. 로그인
로그인 성공 시 "○○○님 반갑습니다!" 출력
2. 회원가입
사용자로부터 값을 받아서 저장
(아이디 중복체크를 할 수 있다면 좋을 듯)
3. 글 목록보기
회원/비회원 모두 볼 수 있다.
제목 - 내용 - 등록(or마지막 수정) 날짜
4. 글 등록
로그인한 회원만 가능하다.
5. 글 수정
본인이 쓴 글만 가능하다.
6. 글 삭제
본인이 쓴 글만 가능하다.
7. 로그아웃
8.회원 탈퇴
9. 프로그램 종료
